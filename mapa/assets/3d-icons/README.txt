Coloca aquí tus modelos 3D (formato .glb, .gltf, .obj, .fbx, etc.) para usarlos como iconos en la vista 3D.

Sugerencia de nombres:
- escenario.glb (escenario principal)
- barra.glb (barra de bebidas)
- wc.glb (baños portátiles)

Puedes descargar modelos gratuitos desde:
- https://sketchfab.com
- https://poly.pizza
- https://www.cgtrader.com/free-3d-models
- https://www.turbosquid.com/Search/3D-Models/free

Para integrarlos, se recomienda usar el formato .glb o .gltf para máxima compatibilidad con Three.js.

Si necesitas ayuda para cargar estos modelos en la app, avísame.
